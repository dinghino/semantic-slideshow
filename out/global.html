HTML Slideshow presenter. This whole app is a style execrise to regain confidence with javascript ES5, jQuery and semantic-ui framework. It's purpose is to have an instrument to create an slideshow using simple HTML pages and javascript configuration files and settings. This can allow a developer to quickly create an insider slideshow to show progress on some code, live while presenting, to have multiple team members create multiple slides that can work together with no issues... The App itself is a central state managed application, that uses App.setState to modify the current state and triggers, if needed, behaviours into the initialized main components (namespaces) It dynamically loads slideshows and scripts from a specified folder.

Primary hooks for the application

Even though not needed for standard use, If you need to programmatically access behaviours, moments and triggers of the application in a custom script you can use these hooks to interface with the App and do stuff conditionally
